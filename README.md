# 🛡️ Pythonic-NetScanner v1.0

A fast and efficient Network Port Scanner built with Python. Designed for security researchers and network administrators to identify open ports and services on a target host.

## ✨ Features
- **Fast Scanning:** Optimized socket timeouts for efficient discovery.
- **Service Detection:** Identifies common services like HTTP, SSH, FTP, and MySQL.
- **Easy to Use:** Supports both command-line arguments and interactive input.
- **Error Handling:** Robust handling for hostname resolution and connection issues.
